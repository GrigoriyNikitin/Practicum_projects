# Foodgram

## 1. Краткое описание проекта

Данный проект «Фудграм» — это cайт (доступен по адресу https://openkittygram.mooo.com/), на котором пользователи могут публиковать свои рецепты, добавлять чужие рецепты в избранное и подписываться на публикации других авторов. Зарегистрированным пользователям также доступен сервис «Список покупок», который позволит создавать список продуктов, которые нужно купить для приготовления выбранных блюд.

## 2. Стек (использованные фреймворки)
